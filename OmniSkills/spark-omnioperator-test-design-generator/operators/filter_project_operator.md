# 过滤投影类算子（Filter/Project）速查卡

> 覆盖WHERE过滤与SELECT投影场景，重点关注条件类型组合、逻辑嵌套及过滤结果集特征。

## 算子基础信息

数据流向=读取类 | 功能语义=过滤投影类 | Omni算子=FilterExecTransformer/ProjectExecTransformer | SQL模式=读取类SQL模式 | 开关=spark.omni.sql.columnar.filter/spark.omni.sql.columnar.project/spark.omni.sql.columnar.projfilter

## 输入因子

| 因子编号 | 因子分类 | 因子名称 | 全量可选值 | 影响权重 |
| :--- | :--- | :--- | :--- | :--- |
| F001 | 核心功能因子 | 过滤条件类型 | 等值(=)/不等值(!=,>,<,>=,<=)/范围(BETWEEN)/集合(IN)/模糊(LIKE)/空值(IS NULL/IS NOT NULL) | 极高 |
| F002 | 核心功能因子 | 逻辑组合 | AND / OR / NOT / 组合嵌套 | 极高 |
| F003 | 条件配置因子 | 开关状态 | 开启(默认) / 关闭 | 极高 |
| F004 | 输入数据特征因子 | 数据类型 | 同写入类F003 | 极高 |
| F005 | 输入数据特征因子 | 过滤结果集 | 全部匹配/部分匹配/无匹配/全NULL | 高 |
| F006 | 输入数据特征因子 | 数据量级 | 空表/小表/中表/大表 | 中 |
