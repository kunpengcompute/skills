## 三、数据插入强制规则

> 生成INSERT语句时必须遵守的数据质量底线，确保测试数据具备充分的覆盖度和代表性。

| 规则项 | 强制要求 | 优先级 |
| :--- | :--- | :--- |
| 行数 | 所有INSERT语句必须插入 **≥20行** 数据（异常测试用例除外） | Level0 |
| 边界值 | 必须覆盖该类型的最大值、最小值 | Level0 |
| 零值 | 必须包含0、0.0、false等 | Level0 |
| NULL值 | 每个字段至少出现一次NULL | Level0 |
| 接近边界值 | 最大值-1、最小值+1等 | Level1 |
| 中文字符 | 补充说明为`<是/否>包含中文`的类型，按实际填写决定是否包含中文测试数据 | Level1 |
| 特殊字符 | STRING/CHAR/VARCHAR类型需包含特殊字符（如!@#等） | Level1 |
| 空字符串 | STRING、CHAR、VARCHAR类型需包含空字符串'' | Level1 |
| 浮点精度值 | FLOAT/DOUBLE需包含极小正值、极大正值、精度边界值 | Level1 |
| 时区边界 | TIMESTAMP需包含时区边界值 | Level1 |
| 嵌套结构 | ARRAY/MAP/STRUCT需包含嵌套结构测试数据 | Level2 |
