# 数据类型检视清单（自检阶段参考）

> 用途：用例生成后的自检阶段参考文档，逐项核查数据类型覆盖是否完整。

---

## 数据类型覆盖检视

- [ ] 数据类型矩阵中所有类型是否按实际支持情况覆盖？
- [ ] 支持的类型是否全覆盖？不支持的类型回退场景是否覆盖？

## 边界值覆盖检视

- [ ] 每种类型的边界值（最大/最小/零值）是否覆盖？
- [ ] 每种类型的接近边界值是否覆盖？
- [ ] NULL值场景是否覆盖？

## 数据质量检视

- [ ] 补充说明含中文的类型，中文场景是否按实际填写覆盖？
- [ ] STRING/CHAR/VARCHAR特殊字符/空字符串是否覆盖？
- [ ] DECIMAL精度边界是否覆盖？
- [ ] TIMESTAMP时区边界是否覆盖？
- [ ] ARRAY/MAP/STRUCT嵌套结构是否覆盖？

## 插入规则检视

- [ ] INSERT数据是否≥20行（异常用例除外）？
